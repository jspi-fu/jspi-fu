### 🔬 研究方向

目前聚焦于 **智能体（Agent）** 的构建与应用，尤其关注如何通过大模型驱动自动化任务、工具调用和多步骤推理。在实践中积累了对主流 AI 应用开发平台（如 **Dify**、**扣子**）的深度使用经验，熟悉从提示词编排、知识库集成到工作流设计的完整链路。同时，对 **LangChain** 框架有一定掌握，能够基于其模块化组件搭建定制化的 Agent 应用，并持续探索更高效、可靠的智能体架构。

### 🌱 个人兴趣

是一名 CSDN 技术博主，喜欢通过写作整理学习路径、分享实战经验，也欢迎同行交流指正。日常开发中偏爱合作型 **vibe coding** —— 在舒适的节奏中专注编码，追求逻辑清晰、可维护性强的实现方式，**希望与AI建立良好的沟通方式而非完全依赖**。
